# Esto (E621)

You've caught me, this is Crystal client for e621.net. If you don't know what that is, this probably isn't for you.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     esto:
       github: watzon/esto.cr
   ```

2. Run `shards install`
